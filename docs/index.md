## 系统原则

- Discord 只发摘要和链接，不塞长文。
- Markdown 是长期知识库，适合搜索、回看、沉淀。
- 本地文件化，不依赖重数据库。
- 小而稳定，能每天跑，不把自己变成赛博盆栽。
